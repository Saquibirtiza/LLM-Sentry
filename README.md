# LLM-Sentry: A Model-Agnostic Human-in-the-Loop Framework for Securing Large Language Models

`HarmfulKB` is a curated dataset containing harmful and hamrless questions used to test the robustness of LLM safeguards. It contains 17,069 harmful questions and 17,406 harmless questions covering a wide range of topics such as healthcare, political science, historical events etc.
